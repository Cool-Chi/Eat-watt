# 🍽️ Eat-watt - 晚餐吃什麼？

這是一個專為解決「晚餐吃什麼」世紀難題而設計的網頁小工具。採用了簡約精緻的 Apple 風格介面，並內建深淺色模式切換，讓你在決定晚餐時也能享受流暢的互動體驗！

## ✨ 精選特色

* **四大互動遊戲**：內建 老虎機 🎰、擲骰子 🎲、翻撲克牌 🃏、幸運輪盤 🎡 四種隨機決定模式。
* **預算篩選系統**：支援 $（平價）、$$（中價位）、$$$（奢華）三種預算標籤，可即時切換與篩選名單。
* **自訂晚餐名單**：可自由新增想吃的食物、隨時修改其預算區間，或刪除不再想要的選項。
* **深淺色模式**：精緻的切換開關，隨系統或心情自由切換亮色與暗色主題。

## 🛠️ 開發技術

* **前端架構**：HTML5、CSS3 (包含 CSS 變數、Flexbox 佈局與多種動態過渡動畫)
* **核心邏輯**：原生 JavaScript (Vanilla JS)、HTML5 Canvas (用於輪盤繪製)、LocalStorage (用於本機資料儲存)

## 📂 檔案結構

* `index.html` - 網頁主結構與配置
* `style.css` - Apple 風格樣式、深暗色模式與遊戲動畫特效
* `script.js` - 老虎機、輪盤、撲克牌與骰子的隨機邏輯與資料流控管
* `README.md` - 專案說明文件
